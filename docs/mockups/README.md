# Design mockups

Static HTML references for Phase 5 UI work. Open in a browser to compare with the live app.

| File | Target route | Description |
|------|----------------|-------------|
| `blog-homepage.html` | `/blog` | Public blog listing and editorial layout |
| `cms-dashboard.html` | `/cms` | CMS sidebar, dashboard, and admin chrome |

These files are **not** served in production. Implement styling in `src/components/site/*` and `src/app/globals.css`.
